# AI Playtesting Learning Track

這個目錄用來執行 `Agent x Gaming x AI Playtesting` 學習路線。

目前階段：Phase 0, Week 1。

## 北極星

短期目標不是做一個很聰明的遊戲 AI，而是做出：

```text
Reproducible Evidence + Risk Insight + QA Workflow Integration
```

也就是讓 agents 穩定玩遊戲、留下可重現證據、找出有價值的 QA 風險。

## Phase 0 目標

先建立三個 mental model：

```text
遊戲 = 規則 + 玩家決策 + 回饋循環
測試 = 有目的地探索風險
AI Playtesting = 用 agents 系統性探索遊戲狀態空間
```

本階段不做大型遊戲、不碰 3D、不先上 RL。只做能支撐後續 Project 1 的基本判斷力。

## Week 1 完成標準

- 讀完 MDA Framework，完成 1 頁筆記。
- 建立遊戲 QA 風險分類表。
- 寫出 AI Dungeon Tester v0 的最小設計 brief。
- 能用一句話說明：agent playtesting 要觀測的是 mechanics 如何產生 dynamics。
